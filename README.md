# COS30018_Group3.5_Option-C


This project implements a stock price prediction system using historical stock data and LSTM neural networks.

## Setup

1. Create and activate a virtual environment:
```bash
conda create -n cos30018 python=3.13
conda activate cos30018
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running

```bash
python stock_prediction.py
```
